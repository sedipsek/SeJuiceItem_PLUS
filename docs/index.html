사전 준비

  • Minecraft Java Edition 1.21.2 ~ 1.21.8
  • Java 21 (JDK/JRE)
  • Fabric Loader 0.17.x+ 설치
  • Fabric API (해당 버전용)

다운로드

  1. Fabric Loader 설치 프로그램을 내려받아 대상 프로필(1.21.8 등)에 설치합니다.
  2. Fabric API (mods 파일)를 내려받습니다.
  3. SeJuiceItem+ 모드 .jar (GitHub Releases) 를 내려받습니다.
Fabric API는 mods 폴더에 같이 넣어야 합니다. 로더만 설치하고 API를 누락하면 모드가 동작하지 않습니다.

설치

  1. 런처에서 Fabric 프로필(대상 버전)로 한 번 실행하여 폴더 구조를 생성합니다.
  2. mods 폴더 위치로 이동:
    • Windows: %APPDATA%/.minecraft/mods
    • macOS: ~/Library/Application Support/minecraft/mods
    • Linux: ~/.minecraft/mods
  3. 다음 파일들을 mods 폴더에 복사합니다:
    • fabric-api-...jar
    • sejuice_itemplus-...jar (본 모드)
  4. 런처에서 Fabric 프로필로 게임을 실행합니다.

적용 & 게임 내 확인

모드가 정상 로드되면 크리에이티브 → 재료(Ingredients) 탭에서 아이템이 보입니다.

추가된 아이템 ID

  • sejuice_itemplus:ramen
  • sejuice_itemplus:tuna_can
  • sejuice_itemplus:medicine_bottle
  • sejuice_itemplus:syringe
  • sejuice_itemplus:bandage
  • sejuice_itemplus:kitchen_knife
  • sejuice_itemplus:first_aid_kit
  • sejuice_itemplus:blue_medicine
  • sejuice_itemplus:green_medicine
  • sejuice_itemplus:red_medicine
  • sejuice_itemplus:narcotic_analgesic (합성 결과)

명령어로 빠르게 확인

/give @s sejuice_itemplus:blue_medicine
/give @s sejuice_itemplus:green_medicine
/give @s sejuice_itemplus:red_medicine
/give @s sejuice_itemplus:narcotic_analgesic

리소스 변경 후에는 F3+T로 리로드하세요.

제작법: 마약성 진통제

양귀비 + 파랑/초록/빨강 약을 모양 제약 없이 조합합니다(Shapeless).

조합 재료

  • minecraft:poppy
  • sejuice_itemplus:blue_medicine
  • sejuice_itemplus:green_medicine
  • sejuice_itemplus:red_medicine

결과: sejuice_itemplus:narcotic_analgesic ×1

참고
  • 이 모드의 약/효과는 게임 내 가상 설정입니다.
  • 제작이 막히면 /reload 후 다시 시도하고, 로그의 recipe 경로와 ID 오타를 확인하세요.

문제 해결

  • 제작이 되지 않음: 데이터팩 경로가 data/<modid>/recipe/... (단수 recipe)인지, result.id 키를 썼는지 확인.
  • 아이템이 안 보임: Fabric API를 mods 폴더에 넣었는지, 모드 버전과 게임 버전이 일치하는지 확인.
  • 툴팁(설명) 안 뜸: 기존에 들고 있던 스택에는 적용되지 않을 수 있음. 새로 받거나 /give로 확인.
  • 리소스 반영 안 됨: F3+T 리로드.
  • 콘솔 확인: latest.log 또는 런처 콘솔의 ERROR 라인을 확인.

개발자 참고(선택)

이 모드는 Minecraft 1.21.2+의 RegistryKey 사전 부여 패턴과 데이터 컴포넌트(LORE) 사용을 따릅니다.

// 등록 패턴(요지): RegistryKey → settings.registryKey(key) → Registry.register
Identifier id = Identifier.of("sejuice_itemplus", name);
RegistryKey key = RegistryKey.of(RegistryKeys.ITEM, id);
Item item = factory.apply(new Item.Settings().registryKey(key));
Registry.register(Registries.ITEM, key, item);

// 레시피(1.21.2+): data/sejuice_itemplus/recipe/narcotic_analgesic.json
{
  "type": "minecraft:crafting_shapeless",
  "ingredients": [
    "minecraft:poppy",
    "sejuice_itemplus:blue_medicine",
    "sejuice_itemplus:green_medicine",
    "sejuice_itemplus:red_medicine"
  ],
  "result": { "id": "sejuice_itemplus:narcotic_analgesic", "count": 1 }
}
